RPS Challenge ✊📄✂️

Rock. Paper. Scissors. Powered by Python, Driven by Fun!

A beginner-friendly Python mini project that lets you play the classic Rock, Paper, Scissors game against the computer. The game randomly generates the computer's choice, determines the winner, and keeps track of your game statistics until you decide to quit.

Features
🎮 Play against the computer
🎲 Random computer choices using Python's random module
🔁 Unlimited rounds
📊 Tracks:
Total rounds played
Wins
Losses
Draws
❌ Quit the game anytime by typing quit
✅ Input validation for invalid choices
How to Play
Run the Python program.
Enter one of the following:
rock
paper
scissor
The computer will randomly choose its move.
The winner of the round is displayed.
Type quit to end the game and view your final statistics.
Technologies Used
Python 3
Random Module
